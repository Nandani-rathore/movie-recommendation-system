# movie-recommendation-system
🚀 Movie Recommendation System

This is a Machine Learning project that recommends movies based on user input using content-based filtering.

---

📌 Features:
- Recommends similar movies
- Uses movie dataset analysis
- Simple and fast prediction system
- Beginner-friendly ML project

---

🛠️ Tech Stack:
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
